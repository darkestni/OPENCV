# OPENCV

OpenCV 图像处理和计算机视觉项目

## 项目简介

本项目包含 OpenCV 相关的实验和示例代码，用于图像处理、计算机视觉任务的学习和实践。

## 目录结构

```
.
├── opencv/           # OpenCV 示例代码
└── README.md
```

## 项目内容

- 图像处理基础操作
- 特征检测
- 目标识别
- 其他计算机视觉任务

## 技术栈

- C++
- OpenCV
- CMake

## 环境要求

- OpenCV 4.x
- C++11 或更高版本
- CMake 3.10+

## 编译和运行

```bash
mkdir build && cd build
cmake ..
make
./opencv_sample
```
